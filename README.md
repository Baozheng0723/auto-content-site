# 臭臭自动化内容站

## 项目结构
```
auto-content-site/
├── content/           # 生成的内容
├── templates/         # HTML模板
├── scripts/          # 生成脚本
├── public/           # 静态文件
└── .github/workflows/# 自动部署
```

## 技术栈
- 静态生成：Python + Jinja2
- 部署：GitHub Pages（免费）
- 自动化：GitHub Actions
- 内容源：AI自动生成

## 内容策略
**主题：** AI工具教程、副业指南、效率技巧
**频率：** 每天自动生成1篇文章
**SEO：** 长尾关键词优化

## 变现
- Google AdSense
- 联盟营销（工具推荐）
- 电子书/课程销售

## 成本
- 域名：~50元/年（可选，先用免费域名）
- 部署：0元（GitHub Pages免费）
- 内容生成：用Moonshot免费额度
