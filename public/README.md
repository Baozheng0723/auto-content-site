# PWA本地使用教程

## 快速开始

### 1. 安装Python
访问 https://www.python.org/downloads/ 下载安装

### 2. 启动PWA
打开终端，进入PWA目录：
```bash
cd pwa
python3 -m http.server 8080
```

### 3. 访问
浏览器打开：http://localhost:8080

### 4. 安装到手机/桌面
- Chrome浏览器 → 点击地址栏"安装"按钮
- Safari浏览器 → 分享 → 添加到主屏幕
